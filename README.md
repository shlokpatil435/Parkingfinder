# Parking Finder App 🚗

An Android application that helps users find nearby available parking spots in real-time.

## Features
- 📍 Live location detection
- 🗺️ Google Maps integration
- 🔄 Real-time parking availability via REST API
- 🔍 Search nearby parking by location
- 📱 Clean and responsive UI

## Tech Stack
- Java
- Android SDK
- Google Maps API
- REST API (Retrofit)
- XML Layouts

## Installation
1. Clone the repo
2. Open in Android Studio
3. Add your Google Maps API key in AndroidManifest.xml
4. Run on emulator or device
